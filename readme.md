# Proyecto de Automatización Web con Selenium y Pytest  
**Entrega Final – Curso de Testing Automation**

# Alumna: Nuria Arnaud

## Propósito del Proyecto
Este proyecto tiene como objetivo aplicar los conocimientos adquiridos hasta la Clase 8 del curso, automatizando flujos básicos de navegación en la web utilizando **Selenium WebDriver** y **Pytest**.  
El sitio elegido para la automatización es [SauceDemo](https://www.saucedemo.com/), una aplicación web diseñada especialmente para prácticas de testing.

A través de este trabajo se busca demostrar la capacidad para:
- Interactuar con elementos web.
- Utilizar estrategias de localización (By.ID, By.CLASS_NAME, etc.).
- Implementar esperas explícitas y validaciones de estado.
- Estructurar un proyecto profesional de testing automatizado.

---

##  Tecnologías Utilizadas
- **Python 3.12+**
- **Selenium WebDriver**
- **Pytest** para la estructura de testing.
- **Pytest-HTML** para generar reportes de ejecución.
- **Git & GitHub** para control de versiones.
- **WebDriver Manager** para gestión automática del driver de Chrome.
